# Chat App Server (Backend)

Node.js backend for real-time chat using Socket.IO.

## Features
- Real-time messaging with Socket.IO
- Image upload support
- REST + WebSocket server

## Tech Stack
- Node.js
- TypeScript
- Socket.IO
- Multer

## Setup
```bash
git clone https://github.com/romelt777/chat-app-server
cd chat-app-server
npm install
npm run dev
